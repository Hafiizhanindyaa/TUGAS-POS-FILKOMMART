TUGAS-POS-FILKOMMART

Program ini merupakan simulasi sistem Point Of Sale (POS) sederhana
untuk toko Filkom Mart menggunakan konsep Pemrograman Berorientasi
Objek (OOP) pada Java.

Fitur yang diimplementasikan:

1. Enkapsulasi (Getter dan Setter)
2. Inheritance (FoodProduct, ElectronicProduct, ClothingProduct)
3. Method Overriding (calculateDiscount)
4. Method Overloading (updateStock dan addItem)
5. Polymorphism menggunakan array Product[]
6. Simulasi transaksi penjualan
7. Laporan total penjualan
8. Menentukan produk terlaris

Program dibuat sebagai tugas kelompok mata kuliah
Pemrograman Berorientasi Objek.
